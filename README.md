# 🚀 Intelligent Sales Forecasting & Inventory Optimization Dashboard

An AI-powered business analytics platform built with **Python, Streamlit, Machine Learning, Pandas, Plotly, and Scikit-Learn** for sales forecasting, inventory optimization, business intelligence, and report generation.

---

## 📌 Project Overview

This project helps businesses analyze historical sales data, forecast future revenue, optimize inventory levels, and generate insightful reports through an interactive dashboard.

The application includes:

* 📤 Data Upload
* 🧹 Data Preprocessing
* 📊 Exploratory Data Analysis (EDA)
* 📦 Inventory Optimization
* 🤖 Machine Learning Model Training
* 📈 Sales Forecasting
* 📄 PDF & Excel Reports
* 🚀 Interactive Dashboard

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/pallepatisudheer-max/Intelligent-Sales-Forecasting-Analytics-Dashboard.git
```

Move into the project folder:

```bash
cd Intelligent-Sales-Forecasting-Analytics-Dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
